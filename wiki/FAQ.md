# Frequently Asked Questions

## General

### What is PKM-Universe PKHeX?
PKM-Universe PKHeX is a customized version of PKHeX with pre-installed plugins, a custom theme, and automatic updates from our repository.

### Is this safe to use?
Yes! PKHeX is open-source software. Our version simply adds plugins and a custom theme - no malware or suspicious code.

### Is this free?
Yes, completely free! If you want to support us, you can donate on [Ko-fi](https://ko-fi.com/pokemonlover8888).

### Will I get banned for using PKHeX?
PKHeX itself cannot get you banned. However, using obviously hacked Pokémon online may result in bans. Always use responsibly.

## Technical

### Which games are supported?
All mainline Pokémon games from Gen 1 (Red/Blue) to Gen 9 (Scarlet/Violet) and Legends Z-A.

### How do I update PKHeX?
The app automatically checks for updates. When available, click the notification to download the new version.

### Where are my settings saved?
Settings are saved in `cfg.json` in the same folder as PKHeX.exe.

### Can I use this on Mac/Linux?
PKHeX is Windows-only. For Mac/Linux, you can use Wine or a virtual machine.

## Plugins

### How do I enable plugins?
Plugins should load automatically. If not:
1. Go to `Options > Settings`
2. Check "Enable Plugin Loading" under Startup
3. Restart PKHeX

### Where is Auto-Legality Mod?
Go to `Tools > Auto Legality Mod` in the menu.

### How do I use the Batch Editor?
Go to `Tools > Data > Batch Editor` and enter your commands.

## Troubleshooting

### PKHeX won't start
- Install [.NET 9.0 Runtime](https://dotnet.microsoft.com/download/dotnet/9.0)
- Extract ALL files from the ZIP
- Run as Administrator if needed

### Plugins not showing
- Check that files exist in the `plugins` folder
- Enable plugin loading in Settings
- Restart PKHeX

### "Legality check failed"
The Pokémon has illegal data. Use Auto-Legality Mod to fix it.
