## What is PKM-Universe PKHeX?

PKM-Universe PKHeX is a customized version of PKHeX with:
- Custom red theme
- Pre-installed plugins (Auto-Legality Mod, Living Dex, Seed Finders)
- Enhanced features
- Automatic updates from PKM-Universe

## Getting Started

1. Download the latest release from [Releases](https://github.com/PKM-Universe/PKHeX-ALL-IN-ONE/releases/latest)
2. Extract the ZIP file
3. Run `PKHeX.exe`
4. Done! Plugins are pre-installed.
