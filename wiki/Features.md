# Features Overview

## Custom Red Theme

PKM-Universe PKHeX features a sleek dark red theme for a modern, professional look.

## Included Plugins

### Auto-Legality Mod (ALM)
Automatically make any Pokémon legal with one click!

**How to use:**
1. Create or import a Pokémon
2. Go to `Tools > Auto Legality Mod`
3. Click `Legalize` or use the hotkey

### Living Dex Generator
Generate a complete Living Pokédex automatically.

**How to use:**
1. Load a save file
2. Go to `Tools > Living Dex`
3. Select your options
4. Click Generate

### Seed Finders
Find seeds for various games:
- **SVSeedFinder** - Scarlet/Violet
- **SWSHSeedFinder** - Sword/Shield
- **PLZASeedFinder** - Legends Z-A

### Spinda Pattern Plugin
Generate specific Spinda patterns.

## Core PKHeX Features

### Save Editing
- Edit any Pokémon in your boxes
- Modify trainer info
- Edit inventory items
- Change game flags

### Showdown Integration
- Import teams from Pokémon Showdown
- Export teams to Showdown format
- Auto-legalize imported sets

### Batch Editor
Edit multiple Pokémon at once using batch commands.

**Example commands:**
```
.Species=Pokemon  # Change species
.IsShiny=true     # Make shiny
.Level=100        # Set level
```

### Mystery Gift Database
Access all event Pokémon and Mystery Gifts.

### Encounter Database
Find wild encounter data for any Pokémon.
